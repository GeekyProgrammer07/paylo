# @paylo/ui

Shared React UI components for the Paylo monorepo.

## Components

- **Appbar** - Navigation header
- **Button** - Styled button component
- **Card** - Container card
- **Center** - Centering wrapper
- **FeatureCard** - Feature display card
- **Footer** - Page footer
- **Icons** - Icon components
- **Select** - Dropdown selector
- **TextInput** - Input field

## Usage

```typescript
import { Button } from "@paylo/ui/button";
import { Card } from "@paylo/ui/Card";

<Card title="Welcome">
  <Button onClick={handleClick}>Click me</Button>
</Card>
```
