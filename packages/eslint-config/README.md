# @paylo/eslint-config

Shared ESLint configurations for the Paylo monorepo.

## Configurations

- **`base`** - Base TypeScript config with ESLint + Prettier + Turbo plugin
- **`next-js`** - Next.js apps with React + Hooks
- **`react-internal`** - React component libraries with Hooks support

Part of the Paylo monorepo
