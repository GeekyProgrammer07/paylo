# @paylo/prettier-config

Shared Prettier configuration for consistent code formatting across the Paylo monorepo.
## Configuration

- **Print Width**: 110
- **Tab Width**: 2
- **Quotes**: Double
- **Semicolons**: Yes
- **Trailing Commas**: ES5
- **Bracket Spacing**: Yes

## Usage

```json
// prettier.config.js
import config from "@paylo/prettier-config/base";

export default config;
```
